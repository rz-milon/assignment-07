Difference between Local Storage and Session Storage
# Local Storage Session Storage
1 Their main difference is that local storage data does not expire whereas session storage data gets cleared when the user closes its browser window.
2 Page sessions are valid for only one tab at a time. So, if a user has three tabs open, each one will have its own page session. Since local storage is not session-based, all the data must be deleted via javascript or manually.
3 The storage capacity of local storage is 5MB/10MB The storage capacity of session storage is 5MB
4 The client can only read local storage The client can only read local storage
5 There is no transfer of data to the server There is no transfer of data to the server
Difference between JavaScript Global and Block Scope
# Global Scope Block Scope
1 The variables defined outside of any function or curly brackets are known as global variables and have global scope. The block scope can be defined as the scope of the variables inside the curly brackets {}.
2 Global scope means that the variables can be accessed from any part of that program, any function or conditional state can access that variable. You are only allowed to refer to these variables from within the curly brackets {}.
